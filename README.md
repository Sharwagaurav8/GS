# 🍽️ GS Restaurant Website

A modern and responsive restaurant website developed using HTML and CSS. This project showcases a complete restaurant landing page featuring food categories, menu items, customer reviews, contact forms, and a mobile-friendly user experience.

## 🚀 Live Demo

https://gs-wheat.vercel.app/

## ✨ Features

- Responsive Navigation Bar
- Attractive Hero Section

- Food Categories Showcase
- Interactive Menu Section
- Customer Testimonials

- Mobile Responsive Design
- Smooth Hover Effects
- Modern Restaurant UI
